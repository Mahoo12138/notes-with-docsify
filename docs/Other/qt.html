零. 食用方案

  1. 大部分项目的代码可分为：
  • // 头文件：该注释标注的代码直接添加到 mainwindow.h 文件顶部的头文件处；
  • // 变量和方法：该注释标注的代码添加到 mainwindow.h 中 class MainWindow类中；
  • // 方法具体实现：该注释标注的代码直接追加到 mainwindow.cpp 中 ；
  • // 构造函数：该注释标注的代码直接覆盖 mainwindow.cpp 中的构造函数；
  1. 部分程序代码是基于Qt的无UI界面工程的，如四至七题，题一是需要Ubuntu虚拟机；
  2. 建议每个工程都参照文档提供的工程，进行本地编译允许，熟悉操作；

一. makefile 编译

用c写程序，源程序分为两个c文件，其中一个定义了一个加法函数，另一个调用这个加法函数，写出这两个 c 文件的源程序，同时写出用于编译的 makefile 文件（写出这 3 个文件，用到的文件名函数名及数字等自拟）；

参考

  1. 打开虚拟机进入文件管理器，创建一下三个文件，并将写入对应代码；
  2. 在文件当前路径打开终端，执行命令 make，编译代码;
